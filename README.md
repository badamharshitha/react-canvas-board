# React Canvas Board

A browser-based drawing application built with React and the HTML5 Canvas API. It includes a toolbar for pen, eraser, line, and rectangle tools, color and brush size controls, clear and undo actions, local storage persistence, and PNG export.

## Getting started

1. Install dependencies with `npm install`.
2. Start the development server with `npm run dev`.
3. Open the Vite URL shown in the terminal to use the app.

## Build

Run `npm run build` to create a production build.
