# 🎨 React Canvas Board

A modern browser-based drawing application built with **React**, **Vite**, and the **HTML5 Canvas API**. Users can create drawings using multiple tools, customize brush settings, save artwork locally, restore previous drawings, undo actions, clear the canvas, and export artwork as PNG images.

---

## 🚀 Features

- ✏️ Pen Tool for freehand drawing
- 🧽 Eraser Tool
- 📏 Line Tool
- ▭ Rectangle Tool
- 🎨 Color Picker
- 📐 Adjustable Brush Size
- ↩️ Undo Last Action
- 🗑️ Clear Canvas
- 💾 Save Drawings to Local Storage
- 🖼️ Artwork Gallery with Restore Functionality
- 📤 Export Canvas as PNG
- ⚡ Fast development using Vite

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/react-canvas-board.git
```

Move into the project folder:

```bash
cd react-canvas-board
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run the Application

Start the development server:

```bash
npm run dev
```

Open the URL displayed in the terminal (usually):

```
http://localhost:5173
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## 🎯 Core Functionalities

- Draw using Pen, Line, Rectangle, and Eraser tools.
- Change drawing color.
- Adjust brush thickness.
- Undo the latest drawing action.
- Clear the canvas instantly.
- Save artwork using Local Storage.
- Restore previously saved drawings from the gallery.
- Export drawings as PNG images.
